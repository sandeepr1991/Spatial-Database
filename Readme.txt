CreateDB.sql
This file contains sql queries needed for creating three tables and populating them. The tables created are HW2_Student, HW2_Building and HW2_Tram.

DropDB.sql
This file contains sql queries required for deleting the tables.

hw2.java

Compiling steps:

Set Class path:
export CLASSPATH=$CLASSPATH:/usr/share/java/mysql.jar

Compile the source code :
javac hw2.java

Now hw2.class file is set up. Now we can run all the queries given in the assignment.
